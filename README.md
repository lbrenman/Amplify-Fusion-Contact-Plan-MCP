# Amplify Fusion Contact Plan MCP

An MCP Server demonstration implemented in Amplify Fusion for generating a sales contact plan for your salesforce active contacts. The contact plan is based on the contact industry and description and generated by a call to OpenAI call.

[Demo video](https://youtu.be/lAPNEqpimOc)

Current tools:

* GetMyCompanyActiveContacts - Get a list of my company's most active contacts that may require attention
* GetContactPlanByContactId - Get a contact plan by contact ID retrieved from the GetMyCompanyActiveContacts tool
